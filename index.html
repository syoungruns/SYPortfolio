About

Full-stack web developer (partial to front-end). Fascinated by patterns in human behavior, design, and connectivity. Natural collaborator, cheerleader, and positive person. Committed to creating code that is not only attractive, but contributes to the greater good for humans and the planet at large. Happiness is cows, vegan donuts, and running in the Seattle rain!

When I'm not playing with divs and p tags, I'm running middle to long distances or trying to bend myself into yoga poses. I love visiting animal sanctuaries, recycling, and devouring smoothies. I consider petting more than one dog on a run a PR, and my dream is to one day own 5 chickens. Fun fact: I grew up in Cereal City, USA. Tech: Ruby on Rails, Bootstrap.

My Contributions: Concept, front-end aesthetic using Bootstrap, back-end functionality. Routes, tie-breaker algorithm, unique results for each user, session, responsive pages.

Challenges: Developing a tie-breaker algorithm that provides a unique result should the user identify with more than one result. Working with Bootstrap for the first time, making sure all pages are responsive. Solo project.
